📡 API Usage

Use this endpoint to generate QR codes programmatically:

http://localhost/generate-qr?url=https://www.google.com

Parameters:

  • url - The URL to encode in the QR code (required)
  • size - QR code size in pixels (optional, default: 300)

Example:

http://localhost/generate-qr?url=https://www.example.com&size=500
